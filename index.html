🚀 DPI Engine Project

Deep Packet Inspection System built in C++

📌 Features

  • ✔ Multi-threaded processing
  • ✔ PCAP file analysis
  • ✔ SNI Extraction (HTTPS detection)
  • ✔ Application Detection (YouTube, Google, etc.)
  • ✔ Packet Blocking System

▶️ Run Command

g++ -std=c++17 -pthread -O2 -I include -o dpi_engine src/dpi_mt.cpp
./dpi_engine test_dpi.pcap output.pcap

✔ Applications Detected: YouTube, Facebook, Google, etc.
